# QuantumWalk-Inspired-Image-Encryption-System

Short description

This repository contains an implementation of a quantum-walk-inspired image encryption system. The primary script `noise.py` generates and applies noise-based transformations to images to demonstrate encryption-like behavior derived from quantum walk concepts.

Contents

- `noise.py` — main script for generating/applying image noise.
- `output/` — example outputs and generated images.

Quick start

1. Create a virtual environment (recommended):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt  # if you have one
```

2. Run the script:

```powershell
python noise.py
```
