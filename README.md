
# QuantumWalk-Inspired-Image-Encryption-System

Description

This project implements a lightweight image encryption demonstration inspired by quantum walk dynamics. It generates a pseudo-random key using a simulated quantum walk, then applies an XOR step followed by S-box (permutation) transformations to encrypt images. The approach is experimental and intended for research, demonstration, and educational use — not production cryptography.

Key features

- Generates a probabilistic key using a quantum-walk-inspired simulation.
- Performs XOR-based masking and row/column permutations (S-box) for encryption.
- Includes visualization helpers to inspect keys, probability distributions, and S-box mappings.
- Example outputs saved under the `output/` directory.

Files

- `noise.py` — main script: provides `encrypt_image`, `decrypt_image`, and visualization utilities.
- `input.jpg` — example input image (if present).
- `output/` — example images generated by the script (encrypted/decrypted/visualizations).
- `requirements.txt` — Python dependencies (added in this commit).

Quick start

1. Create and activate a virtual environment (Windows PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. Run the demonstration (saves `output/full_dashboard.png`):

```powershell
python noise.py
```

Options

You can modify `noise.py` to expose CLI arguments (seed, input file, output folder). I can add a small CLI wrapper if you want.
